#

## GetPhonecodesByRegion (region:String)

:::info
The `GetPhonecodesByRegion` function in react-country-state-city package is used to retrieve a list of [**phone codes**](/1.1.11/types/phonecode) for countries within the given region. This async function typically returns an array of objects containing country names and their corresponding international dialing codes ([**phone codes**](/1.1.11/types/phonecode)). It helps to easily access and display phone codes for different countries in applications.
:::

import PhonecodesByRegionSample from "./phonecodesbyregionsample";

### Usage

```tsx
import React, { useState, useEffect } from "react";
import { GetRegions, GetPhonecodesByRegion } from "react-country-state-city";

export default function App() {
    const [phonecode, setPhonecode] = useState(null);
    const [phoncodeList, setPhonecodeList] = useState([]);
    const [region, setRegion] = useState(null);
    const [regionList, setRegionList] = useState([]);
    useEffect(() => {
      GetRegions().then((result) => {
        setRegionList(result);
      });
    }, []);
    useEffect(() => {
      if (region)
        GetPhonecodesByRegion(region).then((result) => {
          setPhonecodeList(result);
        });
    }, [region]);

    return (
        <div>
        ...
        </div>
    );
}

```

### Parameters

| Name          | Type     | Default | Required | Description                                                                                       |
| ------------- | -------- | ------- | -------- | ------------------------------------------------------------------------------------------------- |
| **region** | `string` | Empty   | ✅       | The **name** of the selected region.                                                             |
| **src**       | `string` | Empty   | ❌       | URL where data files are hosted (e.g., `https://venkatmcajj.github.io/react-country-state-city/`) |

### Results

Returns a [**phone codes**](/1.1.11/types/phonecode) object lists.
