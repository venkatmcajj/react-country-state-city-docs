[1.1.12] - 2025-03-11​

Added​

  • Added a new feature that allows setting a defaultValue for CountrySelect using a name, ID, or country object.

  • Added a new feature that allows setting a defaultValue for StateSelect using a name, ID, or state object.

  • Added a new feature that allows setting a defaultValue for CitySelect using a name, ID, or city object.

  • Added a new feature that allows setting a defaultValue for LanguageSelect using a name, or language object.

  • Added a new feature that allows setting a defaultValue for RegionSelect using a name, ID, or region object.

  • Added a new feature that allows setting a defaultValue for PhonecodeSelect using a country name, or phone_code.
